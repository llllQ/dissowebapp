editFoodItem

This function is called when a user selects a food item from their inventory that they wish to edit details of. This is intiated when a user clicks on the food item's icon. /n The purpose of this function is to bring up a 'edit' html page where the user can edit any of the fields of the selected food item. Upon confirmation, these edits are then written to the current user's inventory in the firebase database. After this occurs the 'edit' html section is hidden and the user is taken back to their inventory page

foodType (any)
