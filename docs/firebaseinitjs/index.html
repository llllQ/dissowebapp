dateDiff

This function is used to calculate the difference between the expiry date and the current date returns that result in the form of days

dateDiff(fromdate: Date, expdate: Date): any
Parameters
fromdate (Date) usually the current date e.g. 08/04/2019
expdate (Date) the expiry date of the food product e.g. 12/04/2019
Returns
any: int value representing number of days until expiry.e.g. 4

writeUserData

This function is called when a new user is registered. It creates an entry for them in the user and inventory tables in the firebase database.

writeUserData(userId: String, displayname: String, email: String)
Parameters
userId (String) current user's id String used to create a flat hierarchy noSQL database entry in firebase
displayname (String) registration field stored in the user table on firebase
email (String) registration field stored in the user table on firebase

readFoodDbData

The readFoodDbData function takes a barcode id and retrieves the relevant food JSON object

readFoodDbData(foodid: String): any
Parameters
foodid (String) where foodid is the String encoded in the 1-dimensional barcode that has just been scanned by the user.
Returns
any: data object representing the item of food that has just had it's barcode scanned.

writeFoodDbData

The writeFoodDbData function uses a barcode id as a key and writes the foodObj data to the firebase database

writeFoodDbData(foodObj: Object, foodid: String)
Parameters
foodObj (Object) is an object depicting a food item (Name, Quantity, Expiry and Inventory Category)
foodid (String) is the String encoded in the 1-dimensional barcode that has been scanned by the user.

writeFoodInvenData

This function receives a object depicting an item of food, and writes it to the current user's firebase inventory database using the current time as a key

writeFoodInvenData(foodObject: Object): any
Parameters
foodObject (Object) a food object containing fields Name (String), Expiry (String) and Quantity (Integer)
Returns
any: boolean value representing if the write request was successful or not
